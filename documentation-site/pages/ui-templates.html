Ontology UI Templates

A demonstration of how complex UI structures are automatically generated from ontology class definitions.

Concept

In a traditional system, every new data type requires a custom form. In WorkFlowProject, we use the ontology as the source of truth. When a user requests a template for a class (e.g., dataset:DataDescriptionFileStaging), the backend parses the ontology properties and returns a recursive JSON structure.

Key Features

  • Recursive rendering of nested objects
  • Type-safe primitive inputs (Keywords, Emails, URLs)
  • Dynamic array handling (Keyword Sets, Lists)

Low Level Overview

Simplifying Scientific Discovery

In scientific research, finding relevant data often depends on how well it is tagged. The Description Extraction system takes a natural language prompt—essentially a researcher's conversational description of a task or dataset—and automatically extracts a concise Title and Keyword.

What is an AI Agent?

An AI Agent is an autonomous system that uses Large Language Models (LLMs) to perform specific tasks. Instead of just chatting, an agent interacts with tools and data to provide structured solutions—in our case, summarizing complex thermodynamics data.

What is Prompt Engineering?

Prompt Engineering is the art of refining instructions (prompts) to get the most accurate results from an LLM. By providing specific roles ("You are a Content Summarization Agent") and strict formatting rules, we ensure the agent returns scientific metadata that is both precise and machine-readable.

Higher Level Overview

The Anatomy of a Prompt

Effective extraction relies on a carefully constructed prompt that grounds the AI agent in its mission. The system uses a Preamble to define boundaries and output formats.

The Summarization Prompt

ROLE: You are a Content Summarization Agent.
TASK: Analyze the USER QUERY and provide a concise title and a single keyword.

RULES:
1. TITLE: Create a brief, descriptive title...
2. KEYWORD: Provide a single keyword... (e.g., 'HeatCapacityCalculation')
3. OUTPUT: Return ONLY a valid JSON object.

FORMAT: {"title": "Summary Title", "keyword": "KeyWord"}

USER QUERY: {query}

Following Google's prompted engineering best practices, we use **few-shot cues** and **role-based grounding** to eliminate ambiguity, ensuring the agent does not hallucinate or include conversational filler in its output.

Deployment Commands

Standardized commands for building and deploying each component.

Angular Frontend (manager-frontend)

# Local npm start # Cloud npm run deploy

Spring Boot Services (Java)

Applies to backend-data-manager and DatasetBackgroundServices.

# Local mvn spring-boot:run # Cloud (Containerize & Deploy) mvn deploy -Pdeploy

Cloud Orchestrator

# Local mvn spring-boot:run # Cloud gcloud run deploy backend-orchestrator --source .

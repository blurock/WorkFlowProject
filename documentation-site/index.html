WorkFlowProject

Architecting an ontology-driven, cloud-native ecosystem for advanced chemical data management. A collection of modular demonstrations that when combined, redefine how research data is captured, processed, and validated.

Ontology UI Templates

Automated interface generation from semantic definitions, ensuring consistent and valid data entry.

RAG (Retrieval-Augmented Generation)

Grounding AI models using formal ontology terms indexed via Google Discovery Engine for precise scientific retrieval.

Value and Term Extraction

Leveraging Large Language Models to map unstructured textual inputs directly into valid ontology structures.

Description Extraction

Automated metadata and abstract generation from source documents using AI-driven analysis.

Fill In Catalog Object

Targeted extraction that automatically populates existing scientific data records with information from text.

Orchestrated Workflows

Coordinating distributed backend services and high-performance computing using Google Cloud Workflows.

Dynamic UI Primitives

An ontology-driven system for recursive user interface generation and complex data management.

Ketcher Molecule Editor

Embedding the open-source Ketcher chemical structure editor in Angular via iframe and postMessage, integrated with the Firestore workflow pattern for backend processing.

Deployment Configuration

Comprehensive overview of local development ports, cloud URLs, and standardized deployment commands across the microservices ecosystem.

About the Project

The WorkFlowProject represents a multi-level documentation effort. It ranges from conceptual overviews for the chemistry community to deep technical dives for software engineers.

The goal is to demonstrate how modern web technologies, AI, and cloud orchestration can be harnessed to create a data management system that is not only powerful but also adhere to the FAIR principles of data stewardship.

Blurock Consulting AB

Founded by Dr. Edward S. Blurock, Blurock Consulting AB specializes in the intersection of chemical research and advanced informatics. With over 35 years of experience, the consultancy drives innovation in FAIR data practices and ontology-driven systems.

JThermodynamicsCloud is the flagship project aiminng to re-implement and enhance thermodynamic calculations in a cloud-native environment, providing unprecedented traceability and accountability in chemical modeling.
